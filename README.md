# SSH 管理面板

> 浏览器即终端，零客户端，全平台。

一个基于 **Node.js** 的 Web SSH 管理工具，让你在任何设备的浏览器里管理服务器。

---

## ✨ 功能

| 功能 | 说明 |
|------|------|
| 🖥️ **SSH 终端** | 浏览器里开终端，支持多标签页 |
| 📁 **文件管理** | 双栏面板，上传下载拖拽操作 |
| 🔒 **安全认证** | HMAC 挑战响应，防止未授权访问 |
| 🔄 **会话保持** | 关闭浏览器也不中断，后台自动保活 |

---

```bash
# 1. 解压
tar -xzf ssh-manager-linux-x64.tar.gz

# 2. 启动
